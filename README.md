# Hello World Flask App

A simple and beautiful Flask web application that greets users by their name.

## Features

- Clean and modern UI with gradient backgrounds
- Responsive design that works on all devices
- Smooth animations and transitions
- Form to input user's name
- Personalized greeting page

## Setup

1. Install the required packages:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
python app.py
```

3. Open your browser and navigate to:
```
http://127.0.0.1:5000
```

## Project Structure

```
hello-world-web-app/
├── app.py              # Main Flask application
├── requirements.txt    # Python dependencies
├── templates/          # HTML templates
│   ├── index.html     # Home page with form
│   └── greet.html     # Greeting page
└── static/            # Static files
    └── style.css      # CSS styling
```

## Usage

1. Enter your name in the input field on the home page
2. Click the "Say Hello!" button
3. You'll be greeted with a personalized message
4. Click "Go Back" to return to the home page
